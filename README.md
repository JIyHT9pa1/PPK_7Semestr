# PPK_7Semestr
Задание по разработке программного комплекса для 7 семестра в технологическом институте. Контрольная работа, вариант 2. Авторы Румянцев Д.К. и Андреев А.К.

Работа с программой. Инструкция.

Описание приложения
Данное приложение предназначено для управления и обработки данных. Пользовательский интерфейс включает в себя форму авторизации, которая обеспечивает доступ к основным функциональным возможностям приложения.


/**********************/
Установка
/**********************/
Для корректной работы приложения необходимо выполнить следующие шаги:

1 Установка MS SQL Server 
Убедитесь, что у вас установлен MS SQL Server. Если у вас его нет, вы можете скачать и установить его с официального сайта Microsoft. Можно скачать SQL management studio.
SQL managament studio https://learn.microsoft.com/ru-ru/ssms/install/install
SQL server https://www.microsoft.com/en-us/sql-server/sql-server-downloads

2 Устновка IDE для редактирования кода
Visual studio https://visualstudio.microsoft.com/ru/

3 Настройка базы данных
Проверьте настройки подключения к базам данных. Приложение должно иметь доступ к необходимым базам данных для корректной работы. Убедитесь, что все необходимые таблицы и данные настроены согласно требованиям приложения. А также настроен корректный путь для базы данных директория\база данных.

В файле Programm.7z распакуйте архив в необходимую директорию. Далее переходите в папку с исполняемой программой. 
Пример после разархивирования
Путь с .exe C:\Users\Administrator\Desktop\Programm - Copy\Programm\TechChemAnalytica\TechChemAnalytica\bin\Debug
Путь до базы данных C:\Users\Administrator\Desktop\Programm - Copy\Programm\TechChemAnalytica\TechChemAnalytica\bin\Debug 

Необходимо открыть файл TechChemAnalytica.exe.config и изменить путь до базы данных со стандартного на C:\Users\Administrator\Desktop\Programm - Copy\Programm\TechChemAnalytica\TechChemAnalytica\bin\Debug\DBMaterial.mdf

4 Сохранить и выйти. 

5 Запустить .exe файл TechChemAnalytica

/**********************/
Авторизация
/**********************/
Для доступа в приложение используйте следующие данные администратора:

Логин: Admin
Пароль: Admin
После введения данных, нажмите кнопку "Войти", чтобы получить доступ к функционалу приложения.
